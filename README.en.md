# Dragon-Li Blog

#### Description
狸花猫博客(Dragon-Li Blog)，是使用微服务架构进行前后端分离的博客系统。Web端使用Vue + Element UI ，后端使用Spring cloud + Spring boot + mybatis-plus进行开发，使用 Jwt + Spring Security做登录验证和权限校验，使用ElasticSearch全文检索服务，
